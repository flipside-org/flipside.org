We are Flipside

Using open source technology, we build web-based tools that help organisations create social impact.

  • Websites

    Build standards compliant websites that help organisations engage with society.

  • Data collection tools

    Use modern technology to allow organisations to conduct surveys and manage their data.

  • Data visualizations

    Make complex issues easier to understand and explore, using open source technology.

  • Collaborative tools

    Help disperse teams to coordinate their work and manage their projects.
